## Project Overview

This project features a comprehensive and interactive business intelligence dashboard created using **Microsoft Power BI**. The goal was to transform raw Zomato restaurant and order data into actionable insights, providing a clear visual overview of key performance indicators (KPIs), trends, and operational details.

The dashboard facilitates data-driven decision-making by summarizing complex data into digestible visual elements.

## Key Features & Visualizations

The dashboard is designed to answer critical business questions, focusing on the following areas:

* **Geographical Analysis:** Visualizations showing restaurant distribution and performance across different cities/locations.
* **Cuisine Popularity:** Analysis of top-performing and most frequently ordered cuisine types.
* **Operational Metrics:** Tracking of sales volume, average order value (AOV), and total revenue over time.
* **Rating and Review Analysis:** Insights into customer satisfaction and factors influencing restaurant ratings.
* **Time-Series Trends:** Visualizing how key metrics change weekly, monthly, or seasonally.

## Technical Stack

* **Tool:** Microsoft Power BI Desktop (`.pbix` file format)
* **Techniques:**
    * **Data Transformation (ETL):** Utilized Power Query for cleaning, shaping, and transforming raw Zomato data.
    * **Data Modeling:** Created an efficient Star Schema (or relevant model) with defined relationships for fast query performance.
    * **DAX:** Implemented custom calculated columns and measures to derive complex metrics (e.g., AOV, YoY Growth).

## How to View

To interact with this dashboard, you need **Microsoft Power BI Desktop**:

1.  **Download Power BI Desktop:** (If you don't have it)
2.  **Clone this repository** to your local machine.
3.  **Open the file:** Open the `Zomato_Insights_Dashboard.pbix` file directly using Power BI Desktop.
4.  **Explore:** Navigate through the different report pages and use the slicers and filters to drill down into the data.
